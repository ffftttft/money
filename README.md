# 基于时空轨迹动力学与多模态语义的道路视频异常预警原型系统

## 项目简介

本项目面向城市道路监控视频，构建了一个无需人工逐帧异常标注的无监督异常风险预警原型。系统通过视觉基础模型提取视频帧特征，再结合统计建模与时间后处理，实现从正常分布建模、异常偏离评分到时序告警输出的完整链路。

核心亮点有两点：

1. 通过高维状态空间中的演化轨迹，观察系统是否向危险区域漂移，实现更早期的风险预警。
2. 在高风险帧上叠加轻量级视觉语言模型，自动生成自然语言解释，便于答辩展示和人工复核。

## 运行流程

1. 按 `docs/SETUP.md` 创建 Python 3.11 环境并安装依赖。
2. 将原始视频放到 `data/videos/`。
3. 运行 `src/data_prep/extract_frames.py` 完成抽帧。
4. 运行 `src/feature_extraction/` 下脚本生成 SigLIP 和 TIMM 特征。
5. 运行 `python src/pipeline/pipeline_2d.py` 生成 2D 对比实验结果。
6. 按需运行 `src/visualization/` 下脚本生成轨迹图、语义解释和展示图。
